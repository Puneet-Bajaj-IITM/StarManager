# StarManager - Inventory Management System

Welcome to StarManager, an inventory management system written in Python. This project is designed to help you manage your inventory by adding, removing, updating, and displaying listings. It also provides insights into the total value of your inventory and the total stock of a particular brand. Furthermore, it allows you to visualize your stock by brand and prices by category.

## Features

- **Add Listings:** You can add new items to your inventory, specifying details such as brand, category, price, and quantity.

- **Remove Listings:** Remove items from your inventory by specifying the unique identifier of the item.

- **Update Listings:** Update the details of existing items, such as modifying the price or quantity.

- **Show Listings:** Display a list of all the items in your inventory.

- **Total Value:** Get the total value of all the items in your inventory.

- **Total Stock:** Find the total stock of a particular brand in your inventory.

- **Visualization:** Generate visualizations to gain insights into your inventory, including stock by brand and price by category.

## Dependencies

To run StarManager, you'll need the following dependencies:

- Python 3.x
- Pandas: A powerful data analysis library for Python.
- Matplotlib: A popular Python plotting library for creating visualizations.
- CSV: A module for reading and writing CSV files in Python.

## Installation

1. Clone the repository to your local machine:

```bash
git clone https://github.com/puneet-bajaj-iitm/starManager.git
cd StarManager
```

2. Install the required dependencies using pip:

```bash
pip install pandas matplotlib
```

## Usage

1. Execute the `StarManager.py` script to start the application:

```bash
python StarManager.py
```

2. Follow the on-screen prompts to add, remove, update, or display listings. You can also generate visualizations and retrieve total values and stocks.

## Project Structure

- `StarManager.py`: The main Python script to run the application.
- `utils.py`: Contains functions for adding, removing, updating, displaying, calculating total values, and generating visualizations.
- `Star.csv`: The CSV file where the inventory data is stored.
